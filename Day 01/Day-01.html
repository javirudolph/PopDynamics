What is R

So, why R? What is R? As the R project website describes it, R is a software environment used for statistics and graphics. However, from personal experience we can tell you that R is much more than that. R is a software environment, and just like many other software you may use to write or make presentations, R was initially developed to perform statistical operations. Nowadays, you can use R to perform statistical analysis, plot your data, make maps, art, and even this tutorial.

Dynamic Equations

  • Represent changes over time in the state of the system
  • Time can be assumed discrete (\(t-1, t, t+1, ...\)) or continuous (\(t>0\))
  • The state variable is a quantity that represents the state of the system at time \(t\) (e.g. \(n(t) =\) population size at time \(t\)).
  • Models in Biology are treated as hypotheses. This is different from physics, where these models are natural laws.

Building dynamic models in continuous time

Ecologists are not mathematicians but recognize that mathematics can be used as a tool to formulate hypotheses.If we are going to consider continuous time dynamic models for species with overlapping generations, we’d better come up with a system that eases the process.Coming up with a mathematical function n(t) that describes the changes in the state of the system (like population size) directly (‘out of the blue’) is difficult.It is easier, however, to think about biological processes that shape the rate of change the system, and then find out if the trends produced by such processes look like the observed trends. When we were talking about annual species, it was easy to build population dynamics models because we only had to predict what was going to happen one time step into the future.In continuous time, we use differential equation, what are these things!!??!!

Differential Equations

What is a differential equation? It is an equation involving an unknown function and its derivatives. For example:

\[\frac{df(x)}{dx} = r f(x)\] where \(f(x)\) is a function of \(x\).
